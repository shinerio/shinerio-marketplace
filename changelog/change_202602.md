## 新增 drawio-diagram-analyst Agent 至 shinerio-note-plugin
date: 2026-02-25
- 将 `drawio-diagram-analyst` agent 从 `.claude/agents/` 迁移至 `plugins/shinerio-note-plugin/agents/`，作为 shinerio-note-plugin 的正式组成部分发布
- 该 agent 可解析 Draw.io 文件的原始 XML 结构，提取架构逻辑、工作流序列和数据关系，生成结构化的文字分析报告
- 支持架构图、流程图、时序图、ER 图、思维导图等多种图表类型，集成 `@next-ai-drawio/mcp-server` 工具
- 更新 `plugin.json` 版本至 1.0.6，同步更新 `marketplace.json` 版本号
- 更新 `README.md` 和 `CLAUDE.md`，补充 drawio-diagram-analyst 的功能说明、使用示例和目录结构
