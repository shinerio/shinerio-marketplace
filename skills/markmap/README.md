## Prerequisites

### mcp

## Usage

## claude code command

create `~/.claude/commands/emd-mindmap.md`.

```text
使用名为"embed-mindmap-into-markdown"的skill为这个markdown文件生成思维导图并嵌入到markdown文件中：$ARGUMENTS
```

## run

`/emb-mindmap [markdown_file_path]`
